## Task（最小执行单元）

- **定义**  
  Task 是整个 WBS 树中的最小执行单元，从 WBS 树中生成并独立执行。

- **主要属性**  
  - **名称 / 需求**  
    描述 Task 的名称或对应的需求。  
  - **步骤与描述**  
    列出完成该 Task 所需的具体步骤和详细说明。  
  - **验收标准 （Acceptance Criteria）**  
    明确判断 Task 是否完成的标准和条件。  
  - **执行人 （Worker）**  
    负责执行该 Task 的角色，可能是 Agent Worker（智能体执行者）或 Human Worker（人工执行者）。
